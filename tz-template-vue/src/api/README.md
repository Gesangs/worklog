# api

> 此目录放置项目接口内容

按资源类型进行文件划分，与后端对应，文件命名与后端资源名称一致。

另外，前端也会拥有自己的资源分类，最常见的如 基础数据 和 用户相关 资源。

```bash
# 目录结构
api/
|── base.js/                  # 基础数据接口
|── user.js/                  # 用户相关接口
|── foo.js/                   # 与后端赌赢的资源接口
|── bar.js/                   # 同上
|
└── ...                       # 其它以上
```
