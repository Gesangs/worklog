# src

## 目录结构

```bash
src/
|── api/                  # 接口
|── assets/               # 图片资源
|── components/           # 项目公用组件
|── icons/                # svg 图标
|── layouts/              # 项目布局组件
|── pages/                # 页面
|── router/               # 路由
|── store/                # 全局状态
|── styles/               # 样式
|── utils/                # 工具函数
|── APP.vue               # 项目页面容器
|── filter.js             # vue 过滤器
|── main.js               # 项目入口
|── mixin.js              # vue 混入
|── permission.js         # 项目权限逻辑
|── plugin.js             # vue 扩展
|── polyfill.js           # 项目垫片函数
```
